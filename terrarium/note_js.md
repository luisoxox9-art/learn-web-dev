# Note

## 什么是 DOM

"Document Object Model"

浏览器一开始根据HTML文档创建了一个**JavaScript对象树型结构**，也就是**DOM**

使用JavaScript通过DOM对Web进行控制

## 我们为什么需要 JavaScript

通过JavaScript修改Web的状态，让Web实现App的功能

## JavaScript 能做什么

能对Web进行修改

## 使用 JavaScript 的一般步骤是什么

通过浏览器提供的API（DOM，CSSOM）修改Web

通过"查询接口"获取Web对象

```js
getElementById()
```

修改Web对象

## "DOM manipulation"是什么含义

通过DOM接口修改Web对象
